# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.1.0]

### Added
- PostHog analytics with privacy controls (opt-out)
- Separate audio/visual notification controls
- Uninstall scripts for CLI and bundled resources
- Recent activity view to track notification events

### Changed
- Generalized copy to support multiple coding agents
- Restructured FAQ with improved alternative solutions comparison
- Sticky navigation and improved FAQ/Support sections

### Fixed
- Sound dropdown functionality and UI accessibility
- FAQ navigation link in Overview section

## [1.0.2]

### Fixed
- Voice sound selection not working

## [1.0.1]

### Changed
- Updated app name to title case: Coding Agent Notifications
- Removed legacy files and consolidated project structure

### Fixed
- Exit code 128 in non-git repositories

## [1.0.0]

### Added
- Initial release
- Smart notification scripts with Claude Code hook integration
- macOS menu bar app (native Swift)
- Tauri-based config editor with one-click installation
- Per-event and per-project sound customization
- Anti-spam cooldown
- Debug logging
- DMG distribution
