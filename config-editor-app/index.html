What are the benefits of using this tool?

Use this tool to receive audio alerts when designated Claude Code events occur. It is intended to:

  • Help ensure that you are alerted as early as possible whenever Claude Code might benefit from your input.
  • Free you from having to actively monitor whether or not Claude Code requires your input.

Some notable technical details

This tool relies on Claude Code hooks to trigger audio alerts. At the time of this writing the following hooks are supported:

  • Notification - runs when Claude Code sends notifications
  • Stop - runs when Claude Code finishes responding
  • PostToolUse - runs after tool calls complete
  • SubagentStop - runs when subagent tasks complete

Customization options

Using the controls below:

  • You can turn audio alerts on and off on per hook type basis.
  • You can arrange for different audio to play on a per hook type basis.
  • You can upload and use custom audio for your audio alerts.

To disable notifications, please set this row's toggle to the "off" position.
